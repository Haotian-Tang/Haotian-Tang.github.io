HyperADRs: Hierarchical Hypergraph Learning for Drug–Gene–ADR Prediction

Pharmacogenomics · Hypergraph Neural Networks · Adverse Drug Reactions

This project develops a hierarchical hypergraph framework for predicting adverse drug reactions by modeling drug–gene–ADR relationships as biologically interpretable hyperedges. The work integrates pharmacogenomic knowledge, causal gene and target information, and pathway-level evidence to improve ADR prediction.

  • Constructed relational hypergraphs over drug–gene–ADR triplets for query-conditioned ADR prediction.
  • Designed ADR categorization strategies to reduce label sparsity and improve physiologically coherent prediction.
  • Benchmarked hypergraph learning models against MLP, GCN, GAT, and HGNN baselines under inductive splits.
  • Coordinated manuscript writing, figure preparation, methods description, and reproducibility materials.

Variant-Level Tensor Modeling for Drug–Disease–ADR Relationships

Tensor Decomposition · Variant-Level Genomics · Pharmacogenetics

This project investigates variant-level genetic effects on drug–disease–ADR relationships using directional co-occurrence matrices and tensor-based modeling. The goal is to quantify how curated pharmacogenomic variants deviate from baseline variant patterns.

  • Integrated PharmGKB toxicity variants with large-scale baseline variant sets for comparative modeling.
  • Constructed variant-indexed directed co-occurrence matrices representing prior and posterior dependencies.
  • Implemented Python pipelines for sparse matrix parsing, cohort-size normalization, and integrity checking.
  • Planned tensor decomposition analyses using Tucker and CP models to evaluate variant-level deviation signals.
